# 🦮 VisionMate AI

> AI-powered assistive system for visually impaired users using Computer Vision, OCR, and Voice Guidance.

  <img src="assets/screenshots/visionmate-overview.png" alt="VisionMate AI Overview" width="100%">

## 🚀 Features

- 🎯 Real-time Object Detection (YOLOv8)
- 🔊 Voice Assistance
- 📍 Left / Center / Right Direction Detection
- 📏 Distance Estimation
- 🚧 Obstacle Detection
- 📖 OCR (Optical Character Recognition)
- ⚡ Multithreaded OCR Processing
- 📷 Webcam Support

---

## 🛠 Tech Stack

- Python 3
- OpenCV
- YOLOv8 (Ultralytics)
- EasyOCR
- Edge-TTS
- NumPy

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/devsharma6/VisionMate-AI.git
```

### Move into the project

```bash
cd VisionMate-AI
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

Windows

```bash
venv\Scripts\activate
```

Linux / macOS

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run

```bash
python main.py
```

## 🎓 Project Purpose

VisionMate AI is being developed as an assistive system to help visually impaired users understand their surroundings through AI-powered vision and speech.

The long-term goal is to provide safe and intelligent navigation using Computer Vision and Artificial Intelligence.
